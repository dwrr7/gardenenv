API Documentation

We provide a JSON-based RESTful API for integration with your system.

  • GET /temperature → Retrieve the current soil temperature
  • GET /moisture → Retrieve the current soil moisture level
  • POST /watering → Activate the automatic watering system
  • GET /status → View the system status in real-time

* Full documentation is available after user account registration.
